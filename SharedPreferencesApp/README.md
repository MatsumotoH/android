SharedPreferencesApp
=======

AndroidのSharedPreferencesを簡単に利用できるクラスを作成しました。  
  
使い方
----------------
保存や呼び出しを行なう際に、いちいちEditorのインスタンスを取得して保存したりを少しでも簡単にまとめられないかなと思い作りました。  
settegerとgetterの呼び出しと簡単に使用できるようになっております。
